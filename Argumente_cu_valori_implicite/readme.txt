

(Argumente cu valori implicite) Un vector n-dimensional este un tablou de n
componente reale. Adunarea a doi vectori se definete astfel:
 u = (x1, ..., xn), v = (y1, ..., yn), u+v = (x1+y1, ..., xn+yn)
Sa se scrie o functie Add cu parametri cu valori implicite, care sa permita adunarea a doi,
trei sau patru vectori. Sa se scrie o functie de test care sa puna în evidenta modul de lucru
al functiei Add (în cazul minimal, se va utiliza aceeai funcie pentru adunarea a 2, 3 i 4
vectori).
